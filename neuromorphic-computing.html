Project Overview

Our neuromorphic computing research focuses on investigating memristor-based architectures that emulate the synaptic behavior of biological neural networks. These novel computing systems leverage the unique properties of memristive devices to achieve brain-like computing capabilities with unprecedented energy efficiency.

By combining advanced circuit simulation with machine learning frameworks, we develop and validate neuromorphic architectures that can perform complex AI tasks while maintaining the low power consumption and adaptive learning capabilities of biological systems.

Key Technologies

Memristor Synapses

Resistive switching devices that emulate biological synapses, enabling analog weight storage and computation with ultra-low power consumption.

  • • Analog conductance modulation
  • • Non-volatile memory storage
  • • Crossbar array architectures

Spiking Neural Networks

Event-driven processing using optical spikes that closely emulate biological neural communication patterns.

  • • Temporal coding schemes
  • • Asynchronous computation
  • • Spike-timing dependent plasticity

Hardware-Software Co-Design

Integrated development approach combining SPICE circuit simulations with TensorFlow models for optimal neuromorphic system design.

  • • Python-based modeling frameworks
  • • SPICE circuit verification
  • • TensorFlow integration
